# ⚡ PowerSyncNerd — AI-Powered Daily Intelligence Pipeline

PowerSyncNerd automatically collects, summarizes, and delivers daily insights on Nigeria's power and energy sector.

## 🚀 Features

- **Intelligent Crawling**: Multi-source news aggregation with deduplication
- **AI Summarization**: LLama-powered content analysis
- **Automated Delivery**: Daily email digests with scheduling
